University Projects

8/2021 – 12/2021 FreedomFI Project, FIU

FreedomFI is designed for areas of the world where the Internet is actively blocked or censored. It is an app that:
- Uses the NetBlocks API
- Provides users with censorship status reports for their connections.
- Details possible remedies and countermeasures.

1/2020 – 5/2020 CyFinder Project, FIU

CyFinder is a plugin for Cytoscape, an open-source platform for visualizing complex networks and integrating them with attribute data. Worked on implementing graph theory algorithms such as:
- Dijkstra’s Algorithm for Shortest Path.
- Kruskal’s Algorithm for a Minimum Spanning
Forest.

1/2020 – 5/2020 Quadrotor Networks Project, FIU

Modified an open source Python Robotics library to set up an environment for an experiment on the Effectiveness of Energy Equalization in Drone Networks
